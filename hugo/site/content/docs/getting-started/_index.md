# Getting Started

Welcome to allbctl! This guide will help you get started with the CLI.

## What is allbctl?

allbctl (aka allbrightctl) is a command-line interface tool for:
- Displaying comprehensive system information
- Bootstrapping development environments
- Detecting and managing development tools
- Tracking git repositories
- Managing database installations
