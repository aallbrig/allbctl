allbctl (aka allbrightctl) is a CLI for Andrew Allbright specific tasks.

Example commands for allbctl:

$ allbctl bootstrap status
$ allbctl bootstrap install
$ allbctl status

Usage:
  allbctl [flags]
  allbctl [command]

Available Commands:
  bootstrap     Manage workstation bootstrap configuration
  completion    Generate the autocompletion script for the specified shell
  help          Help about any command
  list-packages Show package count summary from all detected package managers
  runtimes      Detects programmer development environment runtimes and displays their versions.
  status        Display system information (like neofetch)

Flags:
      --config string   config file (default is $HOME/.allbctl.yaml)
  -h, --help            help for allbctl

Use "allbctl [command] --help" for more information about a command.
